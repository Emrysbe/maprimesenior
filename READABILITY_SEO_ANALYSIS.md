# Analyse de Lisibilité et SEO - Site Monte-Escalier MaPrimeAdapt'

## Résumé Exécutif

**Score de Lisibilité Global**: 6.5/10
**Score SEO Global**: 7/10

### Points Forts
- Structure HTML sémantique correcte
- Balises meta descriptions présentes
- Utilisation cohérente des titres H1-H3
- Appels à l'action présents

### Points à Améliorer

## 1. ANALYSE DE LISIBILITÉ

### 1.1 Longueur des Phrases

**Problème**: Plusieurs phrases dépassent 25 mots, ce qui réduit la compréhension.

#### Exemple Problématique #1 - Hero Section
```
❌ AVANT:
"L'État peut prendre en charge jusqu'à 70% du coût de votre monte-escalier, soit jusqu'à 22 000€ selon vos revenus."
(20 mots - acceptable mais dense)
```

```
✅ APRÈS:
"L'État finance jusqu'à 70% de votre monte-escalier. Le montant peut atteindre 22 000€ selon vos revenus."
(17 mots répartis en 2 phrases courtes)
```

#### Exemple Problématique #2 - MaPrimeAdapt Section
```
❌ AVANT:
"Une aide gouvernementale qui peut couvrir jusqu'à 70% du coût de votre monte-escalier. Vérifiez votre éligibilité en moins de 2 minutes."
```

```
✅ APRÈS:
"L'État prend en charge jusqu'à 70% des coûts. Vérifiez votre éligibilité en 2 minutes chrono."
(Plus direct, plus actif, rythme amélioré)
```

### 1.2 Voix Active vs Voix Passive

**Problème**: Plusieurs passages utilisent la voix passive, ce qui ralentit la lecture.

#### Exemple #1 - Installation
```
❌ AVANT (Passif):
"Nos équipes certifiées installent votre monte-escalier en 24 à 48 heures seulement."
```

```
✅ APRÈS (Actif):
"Nos équipes certifiées posent votre monte-escalier en 24h à 48h."
(Plus court, plus dynamique)
```

#### Exemple #2 - Garantie
```
❌ AVANT:
"5 ans de garantie et un service après-vente réactif pour votre tranquillité d'esprit."
```

```
✅ APRÈS:
"Vous bénéficiez de 5 ans de garantie. Notre SAV répond en 24h."
(Bénéfice direct pour le client)
```

### 1.3 Niveau de Lecture

**Objectif**: Niveau 3ème/Seconde (14-15 ans) pour toucher le public senior et leurs aidants.

#### Simplifications Recommandées

```
❌ Termes Complexes:
- "Assistant à maîtrise d'ouvrage agréé"
- "Plafonds de ressources fixés par l'Anah"
- "Propriétaire occupant"

✅ Alternatives Simples:
- "Expert agréé qui vous accompagne"
- "Limites de revenus définis par l'État"
- "Propriétaire qui habite son logement"
```

### 1.4 Structure des Paragraphes

**Règle d'or**: Maximum 3-4 lignes par paragraphe sur mobile.

#### Exemple - Section "Comment choisir"

```
❌ AVANT (Trop dense):
"Choisir un monte-escalier adapté est une décision importante pour préserver votre autonomie et votre sécurité à domicile. Découvrez les 5 critères essentiels pour faire le bon choix."
```

```
✅ APRÈS (Mieux structuré):
"Choisir le bon monte-escalier préserve votre autonomie.

Votre sécurité à domicile en dépend.

Découvrez les 5 critères essentiels pour réussir votre choix."
```

## 4. RECOMMANDATIONS D'OPTIMISATION TECHNIQUE

### 4.1 Balises HTML Sémantiques

```html
<!-- Ajouter des microdonnées pour les prix -->
<div itemscope itemtype="https://schema.org/Offer">
  <span itemprop="price" content="22000">22 000€</span>
  <span itemprop="priceCurrency" content="EUR">d'aide</span>
</div>

<!-- Ajouter LocalBusiness schema -->
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "LocalBusiness",
  "name": "MaPrimeAdapt' Monte-Escalier",
  "description": "Installation monte-escaliers avec aide MaPrimeAdapt'",
  "areaServed": "FR",
  "priceRange": "€€"
}
</script>
```

## Questions Fréquentes sur les Monte-Escaliers

### Combien coûte un monte-escalier ?
Un monte-escalier droit coûte entre 3 000€ et 6 000€.

L'installation prend 24 à 48 heures maximum.
Pour un escalier droit : 4 à 6 heures.

Oui, absolument.
- Ceinture de sécurité obligatoire
- Détecteurs d'obstacles
- Vitesse limitée à 0,15 m/s
- Arrêt automatique en fin de course

### Puis-je l'utiliser si je suis en fauteuil roulant ?
Un monte-escalier standard nécessite de pouvoir s'asseoir.
Si vous êtes en fauteuil, optez pour une plateforme élévatrice.
MaPrimeAdapt' finance aussi ce type d'équipement.

### L'aide MaPrimeAdapt' est-elle cumulable ?
Oui, avec :
- Les aides des caisses de retraite (500€ à 3 500€)
- Les aides départementales (selon votre région)
- Le crédit d'impôt de 25% (sous conditions)

Total possible : jusqu'à 70% du coût financé.
